# Stock Market Analyzer

A Python data analysis project built using pandas and matplotlib.

## Features

* Analyze an individual stock
* Compare two stocks
* Calculate stock returns
* Display highest and lowest prices
* Calculate average trading volume
* Find the best performing stock
* Find the stocks with the highest average trading volume
* Visualize stock prices with charts

## Technologies Used

* Python
* pandas
* matplotlib

## Dataset

This project uses a stock market dataset containing:

* Date
* Ticker Symbol
* Open Price
* High Price
* Low Price
* Close Price
* Trading Volume

The dataset includes:

* AAPL
* MSFT
* NVDA
* AMZN
* GOOGL
* SPY

## How to Run

1. Install the required packages:

```bash
pip install pandas matplotlib
```

2. Download the repository.

3. Run:

```bash
python analyzer.py
```
